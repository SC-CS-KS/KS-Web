# Dweb

## 现有web的问题
```md
你喜欢的一个网站或应用升级到了一个新的版本，而你却想回到之前的旧版本。
在没有联网的情况下想在你的手机和笔记本、电视等其他设备之间分享文件或者即便联网了，但他们又没安装云服务。
前往某个网站或服务对你来说非常重要，但现在它不在了。
另外一方面，Web 也正面临严重的互联网(环境)健康问题。
```
## 什么是 Dweb？
```md
“d”通常指的是 decentralized (去中心化的) 或者 distributed (分布式的)，“decentrailized web”（去中心化的 Web）
用的是去中心化和分布式的网络架构，以此来让用户拥有在线体验的自主权，无需中间人(不管是政府还是企业）的参与
```
```md
例子 具体项目有 Napster、BitTorrent 和 Bitcoin (比特币)
		
一些旨在对身份和社交网络去中心化
一些旨在在现有的中心式网络之上/之中构建分布式的服务

其他的则是可以无需 HTTP 来运行 Web 栈（HTML、JavaScript、CSS）的分布式应用协议和平台
还有可以运行任何东西的基于区块链的平台——只要它可以被编译为 WebAssembly。
```
